# Product Intel — clawdspiracy.com (patterns to copy)

Source: https://clawdspiracy.com/

This doc captures the **mechanics** that make clawdspiracy a retention + distribution engine. We adapt the mechanics to Mint Machine / Paper Protocol without copying their theme.

---

## What clawdspiracy is (in one sentence)
A **content + status loop** where AI agents produce “case files” about other AI agents, and humans return to watch, rank, and participate.

---

## Core mechanics worth copying

### 1) Narrative framing (adds meaning)
- Everything is themed as an **intel bureau**.
- Content units are “CASE FILES” (not posts).
- People are “AGENTS” and “OPERATIVES” (identity + roleplay).

**Why it works:** transforms ordinary content into a collectible feed.

**Mint Machine adaptation:**
- Convert runs into **"Run Files"** / **"Operator Logs"** / **"Factory Receipts"**.
- Display the player as an **operator identity**.

---

### 2) Identity on the leaderboard
- Leaderboard shows **names + avatars** (Dicebear seeds), not anonymous IDs.
- “Top operatives” section makes the leaderboard feel alive.

**Mint Machine adaptation:**
- Generate a stable display name per sessionId.
- Generate an avatar from sessionId (Dicebear) and show it on leaderboards.

---

### 3) Constant feed of bite-sized artifacts
- A live “case board” shows **latest case files** with timestamps.

**Mint Machine adaptation:**
- Show a live feed of **recent top runs** (score + max combo + time ago).
- Make each run shareable.

---

### 4) Rivalries / faction wars
- Explicit sections for rivalry and wars.

**Mint Machine adaptation (simple v1):**
- “Rivals near you”: pick 3 miners above/below your rank.
- Call-to-action is *one directive*: “Beat Rival_X by +N hash”.

---

### 5) "Summon" mechanic (low-friction creation)
- “Summon an agent”: user enters a topic → unique agent appears.

**Mint Machine adaptation:**
- “Summon a challenge” (later): one-tap daily challenge variants, no forms.
- Or: “Summon a rival” (create a target ghost score to chase).

---

### 6) Quick access / nav that pushes action
- Big, obvious action buttons: Investigations / Comms / Wars / Rivalries.

**Mint Machine adaptation:**
- Nav stays minimal: **Mine / Leaderboard / Prize**.
- Home page should be action-first: **Start Run**.

---

### 7) Disclaimers without killing the vibe
- Disclaimer is present, but not spammy.

**Mint Machine adaptation:**
- Keep Safety/Prize rules pages.
- No modal spam.
